# Modern Web Layouts | HTML & CSS Guide

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A comprehensive, visually-driven demonstration of modern CSS layout techniques. This project showcases the fundamental building blocks of web design through practical, interactive examples.

## 🚀 Live Demo
Experience the live site here: [Modern Web Layouts](https://abdullahkhalil00.github.io/HTML-CSS-website/)

## ✨ Key Features

This project explores four critical areas of frontend layout design:

*   **1. Flexbox Mastery:** Demonstration of one-dimensional layouts, including a responsive navigation bar and a flexible card grid.
*   **2. Advanced Positioning:** A deep dive into `relative` vs `absolute` positioning, showcasing how to create overlapping UI elements and badges.
*   **3. The Box Model:** A visual breakdown of `margin`, `padding`, and `border` interactions, showing how they affect element spacing and size.
*   **4. CSS Grid Layout:** A fully responsive image gallery built with a two-dimensional grid system that adapts seamlessly across mobile, tablet, and desktop.

## 🛠️ Built With

*   **HTML5:** Semantic structure for optimal SEO and accessibility.
*   **CSS3 (Vanilla):** Modern styling without external frameworks, featuring custom properties and smooth transitions.
*   **Google Fonts:** Utilizes *Inter* and *Outfit* for a professional, premium typographic feel.

## 📱 Responsive Design
The website is meticulously crafted to be fully responsive, ensuring a consistent and premium user experience across all devices, from small smartphones to large desktop monitors.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/abdullahkhalil00/HTML-CSS-website.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd HTML-CSS-website
    ```
3.  **Open in Browser:**
    Simply open `index.html` in your preferred web browser.
